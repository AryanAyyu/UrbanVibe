# UrbanVibe

Full-stack e-commerce application.

## Stack
- Backend: Node.js, Express, MongoDB/Mongoose, JWT, Bcrypt, Multer, Cloudinary, MVC
- Frontend: Vite (React), Tailwind CSS, Redux Toolkit, React Router, Axios

## Structure
```
UrbanVibe/
  backend/
  frontend/
```

## Backend setup
```
cd UrbanVibe/backend
npm install
npm run dev
```
Create `.env` in backend (already scaffolded):
```
PORT=5000
MONGO_URL=yourMongoUri
JWT_SECRET=yourSecretKey
CLOUDINARY_CLOUD_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=
```

## Frontend setup
```
cd UrbanVibe/frontend
npm install
npm run dev
```
